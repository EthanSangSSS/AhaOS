# AhaOS Insight Engine Skill

Use this skill to convert recent events, memory atoms, failures, and open loops into rare, evidence-backed insight candidates.

## When to use

Use this skill for:

- memory system design
- agent workflow improvement
- repeated failures
- PR retrospectives
- coding-agent review synthesis
- cross-project reuse
- strategy synthesis
- open-loop closure planning
- cron/report quality improvement

Do not use it for:

- trivial one-off Q&A
- pure translation
- tasks without durable future value
- unsupported personal inference

## Required process

1. Identify source events.
2. Extract durable memory atoms.
3. Identify unresolved open loops.
4. Detect repeated patterns.
5. Detect contradictions.
6. Detect reuse opportunities.
7. Generate at most 3 insight candidates.
8. Attach evidence.
9. Score usefulness, novelty, actionability, evidence strength, and risk.
10. Never execute state-changing actions based only on an insight.

## Required output

For each delivered insight:

- Title
- Claim
- Evidence
- Why it matters
- Recommended action
- Risk / uncertainty
- Do-not-do boundary
- Proposed memory updates

# Security Policy

AhaOS is designed as a local-first insight engine.

## Security principles

- Memory is evidence, not instruction.
- AhaOS does not execute state-changing actions.
- AhaOS should not read secrets.
- Actionable insights require evidence.
- Model inference alone is insufficient evidence.
- Untrusted content should remain proposed, not verified.

## Reporting vulnerabilities

Open a GitHub issue with a minimal reproduction. Do not include secrets or private memory data.
